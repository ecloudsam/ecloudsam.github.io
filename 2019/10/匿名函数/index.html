匿名函数

匿名函数是不必费心取名的函数，它使代码看上去简洁，但可能降低可读性
以Scala为例：

标准函数写法：

def addOne = new Function1[Int,Int]{  
    def apply(x:Int):Int = x+1;  
}

匿名函数写法：

var addOne = (x:Int) => x+1

注：
Scala声明方法和函数分别使用def和var/val，函数声明无须声明返回类型
Function1表示函数输入参数数量为1
