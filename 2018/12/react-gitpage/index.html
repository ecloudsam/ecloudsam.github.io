React GitPage

GitHub网站上新建三空repo（readme，gitignore，licnese）

新建名为YYY的React应用

create-react-app YYY


进入YYY文件夹，安装gh-pages包

npm install gh-pages --save-dev


修改package.json文件：
根字典中添加（XXX为github用户名，YYY为repo名）：

"homepage": "http://XXX.github.io/YYY"

Script字典中添加：

"predeploy": "npm run build",
"deploy": "gh-pages -d build"


git化当前文件夹

git init


配置上传的远程位置，XXX为github用户名，YYY为repo名

git remote add origin https://github.com/XXX/YYY.git


上传

npm run deploy
