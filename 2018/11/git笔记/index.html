GIT笔记

市面上的git教程确实又多又全，但对于新手，很多时候他们想要的只是一份——傻瓜操作手册

查看Git配置

git config --list


配置Git帐号

git config --global user.name "XXX"  
git config --global user.email "XXX@XXX.XXX"


生成密钥（将id_rsa.pub中的内容复制到GitHub的SSH Key中）

ssh-keygen -t rsa -C "XXX@XXX.XXX"


测试是否成功连接到GitHub

ssh -T git@github.com


git化当前文件夹

git init


配置上传的远程位置，XXX为github用户名，YYY为repo名

git remote add origin git@github.com:XXX/YYY.git


（上一步输入错误时）移除当前设置的上传目录

git remote rm origin


暂存本地所有改动

git add -A


认可本地所有改动，备注为none

git commit -a -m "none"


上传

git push


强制push

git push -u origin master -f
